Proyecto Flask_OCR

Esta aplicación web, construida con el framework Flask, permite a los usuarios realizar reconocimiento óptico de caracteres (OCR) en imágenes cargadas.

Componentes Clave

  • Importaciones: Se utilizan bibliotecas como Flask, PIL, pytesseract, sqlite3 y otras para las funcionalidades de la aplicación.
  • Configuración: Se establecen parámetros como la ubicación de las imágenes cargadas, formatos permitidos y claves secretas.
  • Base de datos: Se utiliza SQLite para almacenar los resultados del OCR y la información de los usuarios.
  • Manejo de usuarios: Se implementa un sistema de autenticación para permitir el acceso de los usuarios a diferentes funcionalidades.
  • OCR y procesamiento de imágenes: Se utiliza pytesseract para extraer texto de las imágenes y se almacenan los resultados en la base de datos.
  • Interfaz de usuario: Se crea una interfaz web para cargar imágenes, ver los resultados del OCR y administrar los usuarios.

Funcionalidades Principales

  • Cargar imágenes y realizar OCR.
  • Almacenar los resultados del OCR en una base de datos.
  • Visualizar los resultados del OCR.
  • Editar y eliminar resultados.
  • Autenticación de usuarios.

Tecnologías Utilizadas

  • Flask
  • Python
  • SQLite
  • PIL
  • pytesseract
  • Jinja2 (Probablemente)
